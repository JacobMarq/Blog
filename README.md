# The Blog-Spot

A blog app with Devise User Authentication
     
     User accounts can 
      -post articles/comments
      -like articles/comments
      -dislike articles/comments
      -favorite articles
      -follow users

Ruby version ~3.0.2

Rails version ~ 6.1.4

Avoid ffi_c not found error:
gem install ffi --version '1.15.5' -- --disable-system-libffi
